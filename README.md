# Cona Footwear Website (React)

A clean, responsive ecommerce-style enquiry website for **Cona Footwear**.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Features

- React + Vite implementation
- Responsive pages: Home, Products, About, Contact
- Black/white/gold premium theme
- Click-to-call and WhatsApp enquiry actions
- Google Maps embed for Navrangpura, Ahmedabad
